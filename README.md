## creytm
Just a simple and **illegal** MPEG-3 downloader for YouTube videos. [Demo](https://creytm.vercel.app/aPnqx56V8-0)

### Usage
```http
GET https://creytm.vercel.app/<YT-ID>
```

> [!IMPORTANT]
> Do not include the following:
> 
> 1. Parameters <br>
> 2. Hash (`#...`) <br>
> 3. `/` at the end
